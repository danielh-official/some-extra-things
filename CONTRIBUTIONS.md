# Contributions

Open an issue to make a request or report a bug. I'll check and accept it if I want there to be any pull requests for it.

## Setup

1. Fork and clone the repository
2. Follow [NativePHP Desktop Development Documentation](https://nativephp.com/docs/desktop/2/getting-started/development) to get started.

## Making a Pull Request

Pull requests will only be accepted if there is an accepted issue tied to them or if I've made them.
